BREVO API OTP SETUP

Use this update when SMTP gives ETIMEDOUT or connection timeout on Render.

Render Environment Variables:
1. Add this new variable:
   BREVO_API_KEY = your Brevo API key

2. Keep this variable:
   SMTP_FROM = Vasuki NFC <vasukinfc@gmail.com>

3. SMTP_HOST, SMTP_PORT, SMTP_SECURE, SMTP_USER and SMTP_PASS are no longer required for OTP email when BREVO_API_KEY is set. You can leave them or remove them.

How to get BREVO_API_KEY:
Brevo Dashboard -> top-right account/settings -> SMTP & API or API Keys -> Generate a v3 API key -> copy it to Render.

After adding BREVO_API_KEY:
Render -> Save, rebuild, and deploy -> test register -> check Brevo Transactional Logs.
